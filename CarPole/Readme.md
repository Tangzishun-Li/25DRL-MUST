## 1. Virtual Environment

- Create      
  
  > python -m venv CartPole

- Activate:   
  
  > source CartPole/bin/activate
  
  **(For Windows OS)**
  
  > CartPole\Scripts\activate.bat 

- Deactivate: 
  
  > deactivate

- Check:      
  
  > which python
  
    **(For Windows OS)**
  
  > where python

## 2. Install required packages

> pip install -r requirements.txt

## 3. Run program

> python CartPole_DQN.py
> python CartPole_SARSA.py
> python CartPole_A2C.py
